Questo è un FIX UI / VISUAL.

Principi inviolabili Aurelius:
— POTENTE nei dati
— SEMPLICE nell'interfaccia
— BELLA nel design
— FUNZIONALE per tutti gli utenti
  (25 anni e 70 anni)

Obiettivo:
— correzione minima e chirurgica
— leggibilità migliorata
— coerenza con il design system
— zero allargamento a backend o
  database non richiesto

Prima di modificare, dimmi sempre:
1. Modifica UI proposta
2. File coinvolti
3. Token di design usati
   (colori, font, spacing)
4. Cosa NON verrà toccato

Regole:
— usa sempre AureliusTheme.*
  invece di colori hardcoded
— tap target minimo 44x44px
— testo minimo 13px
— sempre ▲▼ per gain/loss
  (non solo colore verde/rosso)
— Privacy Mode deve funzionare
  su ogni valore numerico modificato
— dark/light mode devono funzionare
  entrambe dopo ogni modifica
— non toccare servizi o provider
  Riverpod se non necessario
